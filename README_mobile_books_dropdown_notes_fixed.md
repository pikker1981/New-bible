# Mobile books dropdown + glossary readability patch

## Apply
Copy these files into the project root and overwrite existing files where prompted.

- app.js
- index.html
- mobile_books_dropdown_notes_fixed.css
- data/manifest.json

## Changes
- Mobile BOOKS selector is now a dropdown-style scroll list.
- Book options show English book names with Korean/order metadata.
- Glossary note items show both title and description preview again.
- Long glossary text wraps instead of clipping to the right.
- Existing 27-book search, notes, and highlight marking logic are preserved.
