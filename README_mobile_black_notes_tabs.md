# Mobile black selector + note tabs patch

## 적용 파일
- `app.js`
- `index.html`
- `mobile_black_notes_tabs.css`
- `data/manifest.json`은 기존 27권 manifest 유지용으로 포함했습니다.

## 변경 내용
- 모바일 권 선택 버튼을 권별 컬러가 아닌 블랙/화이트 톤으로 통일
- 모바일 장 선택 active 버튼도 블랙 톤으로 통일
- 모바일에서 `SELECTED NOTE`를 장 선택과 본문 사이로 이동
- `설명 리스트`는 접이식 탭으로 분리
- 데스크톱 레이아웃은 기존 좌/우 레일 구조 유지
