# Desktop BOOKS abbreviation + SEARCH position patch

## 적용 파일
- app.js
- index.html
- mobile_books_dropdown_select_fixed.css
- data/manifest.json

## 변경 사항
1. 데스크톱 BOOKS의 권 약자를 중복 권명 기준으로 수정했습니다.
   - 고전, 고후, 살전, 살후, 딤전, 딤후, 벧전, 벧후, 요1, 요2, 요3
2. SEARCH 패널을 BOOKS 패널 바로 위로 이동했습니다.
3. 기존 모바일 BOOKS 드롭다운, 전체 27권 검색, 각주, 형광펜 기능은 유지했습니다.
