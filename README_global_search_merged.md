# 전체 27권 검색 통합 패치

## 적용 파일

- `app.js` → 기존 루트의 `app.js`에 덮어쓰기
- `data/manifest.json` → 기존 `data/manifest.json`에 덮어쓰기
- `index.html` → 참고용으로만 포함. 기존 파일과 구조 변경 없음

## 변경 내용

- 기존 검색: 현재 선택된 권 안에서만 검색
- 변경 검색: `manifest.json`에 등록된 27권 전체에서 검색
- 검색 결과는 권 순서대로 표시
- 검색 결과의 왼쪽 참조 영역을 누르면 해당 권/장/절로 이동
- 기존 형광펜 마킹, 각주 표시, 권별 포인트 컬러 로직 유지

## 주의

로컬 테스트는 `index.html` 직접 열기가 아니라 아래처럼 서버로 실행해야 합니다.

```bash
python -m http.server 8000
```
